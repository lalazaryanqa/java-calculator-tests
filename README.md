# Java Calculator Tests

Simple Java console calculator project with basic arithmetic operations and TestNG unit tests.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulo operation
- Scanner user input
- Console application
- Basic exception handling

## Technologies

- Java
- Maven
- TestNG
- Git & GitHub
